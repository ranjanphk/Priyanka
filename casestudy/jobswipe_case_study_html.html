Problem Statement

Job searching is slow, repetitive, and inefficient. Companies receive irrelevant resumes, while candidates waste time browsing listings. There is no fast, intuitive matching mechanism.

Goal

  • Create a Tinder-style job discovery experience.
  • Enable recruiters to swipe on candidates.
  • Make job discovery fast and engaging.

User Flow

Candidate Flow

Onboarding → Skill Selection → Job Feed → Swipe → Match → Chat

Company Flow

Login → Candidate Cards → Swipe → Match → Connect

Key Features

  • Swipe-to-Apply: Right swipe = Interested, Left swipe = Skip
  • Real-Time Notifications: Companies get candidate cards instantly
  • Company Swipe System: Recruiters select or reject with a swipe
  • Instant Match & Chat for mutual right swipes
  • Dark Mode UI fully designed with Tailwind
